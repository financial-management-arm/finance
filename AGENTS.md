# Codex Working Rules

1. Work only on the user's explicit request.
2. Inspect only files directly relevant to that request.
3. Prefer the smallest correct edit.
4. Use no more than 8 tool calls unless the user approves more.
5. Do not delete, move, restore, rename, or reorganize files without explicit approval.
6. Do not modify Git state, branches, commits, remotes, or deployments unless explicitly requested.
7. Do not browse the internet unless required by the request.
8. Do not perform unrelated cleanup, redesign, refactoring, or investigation.
9. Run one focused verification after editing, then stop.
10. Ask one concise question only when proceeding would risk destructive or incorrect work.
11. Keep progress updates and final responses brief.
